# MiniExhibitionHall 场景搭建指南

> 跟着做，零基础也能搭出展览馆

---

## 步骤1：创建 Unity 项目

1. 打开 Unity Hub → 新建项目
2. 选择 **"3D Core"** 模板
3. 项目名设为 `MiniExhibitionHall`，路径选 `G:\MiniExhibitionHall`
4. 点击创建

---

## 步骤2：导入框架脚本

1. 在项目窗口的 `Assets` 下右键 → Create → Folder，依次创建：
   - `Scripts/Player`
   - `Scripts/Exhibition`
   - `Scripts/UI`
   - `Scripts/Audio`
   - `Scenes`
   - `Prefabs`
   - `Materials`
   - `Audio`
2. 将所有 `.cs` 脚本文件拖入对应文件夹

---

## 步骤3：搭建展览馆场景

### 3.1 创建地面
- Hierarchy 右键 → 3D Object → Plane
- 改名为 `Floor`，设置 Scale 为 (10, 1, 10)
- 创建新 Material，命名为 `Mat_Floor`，颜色选浅灰色或木纹色，拖给 Floor

### 3.2 创建墙壁
- 3D Object → Cube，改名为 `Wall_North`
- Scale 设为 (20, 3, 0.2)，Position 设为 (0, 1.5, -5)
- 同样方式创建 Wall_South、Wall_East、Wall_West
- 也可以做一个天花板 (Cube, Scale 20x0.2x10)

### 3.3 创建展台
- 3D Object → Cube，改名为 `Pedestal_1`
- Scale 设为 (0.6, 0.8, 0.6)，Position 可放在展区中心
- 可添加一个薄顶板 (Cube, Scale 0.8x0.05x0.8) 放在上面做台面
- 复制多个展台分布在不同位置

### 3.4 设置光照
- 保留默认的 Directional Light
- 在各展台附近添加 Point Light（右键 → Light → Point Light）
- Point Light 的 Color 选暖黄色 (#FFF3E0)，Intensity 0.8，Range 5
- Window → Rendering → Lighting → Environment → 可调整 Ambient Color 为深灰蓝色营造氛围

---

## 步骤4：设置玩家

### 4.1 创建玩家 GameObject
1. Hierarchy 右键 → Create Empty，命名为 `Player`
2. 为 Player 添加组件：
   - **Character Controller** (Component → Physics → Character Controller)
     - Center: (0, 1, 0) / Height: 2 / Radius: 0.3
   - **FirstPersonController** (把脚本拖上去)
3. 在 Player 下创建子物体：右键 Player → 3D Object → Camera
   - Camera 的 Position 设为 (0, 0.6, 0)【眼睛高度】
4. 为 Player 添加 Tag：在 Inspector 顶部 Tag 下拉中选择 "Player"（如果没有，选 Add Tag 创建一个）

### 4.2 测试一下
点击 Unity 顶部 Play 按钮 → 鼠标锁定 → WASD 移动 → OK!

---

## 步骤5：添加展览品

### 5.1 放置展品模型
1. 导入你的3D模型（拖入 Assets 窗口）或使用 Unity 内置几何体：
   - Sphere（球体）→ 可代表行星
   - Cube + Cylinder 拼合 → 可代表火箭/展台
2. 将模型拖到场景中展台上方
3. 调整 Position 和 Scale 到合适大小

### 5.2 配置展品脚本
1. 选中展品 → Add Component → ExhibitBase（或 ExhibitPickup）
2. 在 Inspector 中填写：
   - **Exhibit Name**: 展品名称（如"火星探测器模型"）
   - **Exhibit Description**: 展品介绍（支持多行文字）
   - **Hint Text**: "按 E 查看详情"
   - **Interaction Distance**: 3（默认即可）
   - **Auto Rotate Speed**: 10（展品会缓缓旋转）
3. 添加碰撞体：Add Component → **Sphere Collider**（或Box Collider）
   - 勾选 **Is Trigger**
   - 调整 Radius/Size 使其覆盖展品周围3米的区域

### 5.3 复制更多展品
- 用 Ctrl+D 复制，修改名称和描述，放不同位置
- 建议 4-6 个展品分布在 2-3 个展区

---

## 步骤6：设置 UI

### 6.1 创建 UI Canvas
1. Hierarchy 右键 → UI → Canvas
2. Canvas 的 Render Mode 设为 **Screen Space - Overlay**

### 6.2 创建信息面板
在 Canvas 下创建：
1. 右键 Canvas → UI → Panel，命名为 `InfoPanel`
   - 设置锚点在屏幕中央
   - 大小约 500x400
   - 背景色半透明黑色
2. 在 InfoPanel 下创建：
   - UI → Text，命名为 `TitleText` —— 展品标题（字体大、加粗）
   - UI → Text，命名为 `DescText` —— 展品描述（字体正常）
   - UI → Button，命名为 `CloseBtn` —— 关闭按钮
     - Button 的 OnClick 事件拖入 GameManager，选 `HideExhibitInfo`

### 6.3 创建交互提示
- Canvas 下创建 UI → Text，命名为 `HintText`
- 锚点放在屏幕底部中央
- 字体大小适中，颜色白色
- 文字内容在代码中动态设置

### 6.4 创建开始面板
- Canvas 下创建 UI → Panel，命名为 `StartPanel`
- 添加 Title 文字："欢迎来到虚拟展览馆"
- 添加 Button "开始参观"，OnClick → GameManager.StartGame
- 添加 Button "退出"，OnClick → GameManager.QuitGame

### 6.5 绑定 UI 引用到 GameManager
1. 在 Hierarchy 中创建空对象，命名为 `GameManager`
2. Add Component → GameManager
3. 将所有 UI 引用拖入 Inspector 对应槽位：
   - Start Panel → StartPanel
   - Info Panel → InfoPanel
   - Title Text → InfoTitleText
   - Desc Text → InfoDescText
   - Hint Text → InteractionHint

---

## 步骤7：设置音频

1. 在 Hierarchy 中创建空对象，命名为 `AudioManager`
2. Add Component → AudioManager
3. 将下载的 BGM 音频文件拖入 BGM Clip 槽位
4. 将环境音文件拖入 Ambient Clip 槽位
5. 调整 BGM Volume 和 Ambient Volume

---

## 步骤8：搭建展区（可选进阶）

### 为每个展区创建 ZoneTrigger
1. 创建空对象，命名为 `Zone_AreaA`
2. Add Component → Box Collider，勾选 Is Trigger
3. 调整 Size 覆盖整个展区范围
4. Add Component → ZoneTrigger
5. 填写：
   - Zone Name: "古代文明展区"
   - Zone Description: 展区的文字介绍
6. 复制创建多个区域

---

## 步骤9：保存场景

- Ctrl+S 保存场景到 `Assets/Scenes/` 文件夹
- 命名为 `MainScene`

---

## 步骤10：打包发布（可选）

1. File → Build Settings
2. 添加当前场景到 Scenes In Build
3. Platform 选 PC, Mac & Linux Standalone
4. 点击 Build，选择输出文件夹
5. 生成 `.exe` 可执行文件

---

## 调试技巧

| 问题 | 解决方法 |
|------|----------|
| 玩家穿过墙壁 | 给墙壁添加 Collider（不要勾选 Is Trigger） |
| 展品无法交互 | 检查 Player Tag 是否为 "Player"，展品 Collider 是否勾选 Is Trigger |
| UI 不显示 | 检查 Canvas 的 Render Mode，检查 GameManager Inspector 中引用是否正确 |
| 鼠标无法锁定 | 确认按了 Play 后点击了 Game 窗口 |
| 拾取后展品位置不对 | 调整 ExhibitPickup 的 Pickup Offset 和 Pickup Distance |
