# 太空探索展 - 场景搭建指南

> 主题：太空探索虚拟展览馆 | 展品数量：6件 | 展区数量：3个

---

## 快速开始（一键搭建）

1. 在 Unity 中打开项目
2. 点击顶部菜单 **Tools → 搭建太空探索展览馆**
3. 在弹出的窗口中点击 **🚀 一键搭建太空展览馆**
4. 等待生成完成
5. 点击 **Play** 开始体验

> 如果场景中已有物体，建议先点击 **🗑 清除现有场景物体**。

## 3I 特性分析（用于项目报告）

### Immersion 沉浸性
- 完整封闭的展览馆空间（墙壁+天花板+地面）
- 三色主题灯光营造太空氛围（蓝/橙/紫）
- 暗色调环境光模拟太空深邃感
- 展品自动旋转展示
- 第一人称视角自由探索

### Interaction 交互性
- 6件展品支持靠近查看信息面板
- 5件展品支持拾取后360度旋转观察
- 鼠标拖拽旋转 + 滚轮缩放
- 3个展区自动触发区域介绍
- 实时UI反馈（提示文字 + 信息面板）

### Imagination 构想性
- 在虚拟空间中"触摸"火箭和卫星
- 近距离观察土星光环和火星陨石坑
- 查看深空探测器上的金唱片
- 突破物理限制的观展体验

## 手动搭建步骤（不使用一键工具时参考）

### 步骤1：创建项目
- Unity Hub → 新建 3D Core 项目

### 步骤2：导入脚本
- 将 `Assets/Scripts/` 下所有 `.cs` 文件放入项目对应目录
- 注意 `Editor/` 文件夹下的脚本需放在 `Assets/Editor/` 中

### 步骤3：运行一键搭建
- Tools → 搭建太空探索展览馆

### 步骤4：手动补充（可选）
- 在 AudioManager 的 Inspector 中拖入 BGM 音频文件
- 在展品的 Inspector 中拖入专属音效文件
- 调整光照参数微调氛围

## 展品详细配置参考

### 长征火箭模型
- **3I-沉浸**：火箭发射场景还原了真实发射的紧张与震撼
- **3I-交互**：靠近观看火箭结构细节，按E查看详情
- **3I-构想**：通过虚拟展览近距离接触国之重器
- **自动旋转**：15°/秒
- **可拾取**：是

### 人造卫星模型
- **3I-沉浸**：卫星在太空中缓缓旋转，模拟轨道运行
- **3I-交互**：按E拾取卫星，360度观察太阳能板结构
- **3I-构想**：体验在太空中近距离观察卫星的奇妙感受
- **自动旋转**：20°/秒
- **可拾取**：是

### 火星
- **3I-沉浸**：置身红色星球表面，感受异星世界的荒凉与壮美
- **3I-交互**：按E拾取火星模型，旋转观察其标志性地貌
- **3I-构想**：想象人类未来在火星建立城市的场景
- **自动旋转**：10°/秒
- **可拾取**：是

### 土星
- **3I-沉浸**：土星光环缓缓旋转，营造梦幻的太空氛围
- **3I-交互**：按E拾取模型，从不同角度欣赏光环的壮丽
- **3I-构想**：在虚拟空间近距离欣赏天文奇观
- **自动旋转**：10°/秒
- **可拾取**：是

### 国际空间站模型
- **3I-沉浸**：模拟空间站内部环境，体验宇航员工作场景
- **3I-交互**：按E拾取，拖拽旋转查看各舱段结构
- **3I-构想**：构想未来大型太空城市的蓝图
- **自动旋转**：8°/秒
- **可拾取**：是

### 深空探测器模型
- **3I-沉浸**：感受探测器在无尽深空中航行的孤独与壮丽
- **3I-交互**：按E查看探测器的科学仪器和金唱片信息
- **3I-构想**：想象外星文明收到人类问候的那一刻
- **自动旋转**：15°/秒
- **可拾取**：否（仅查看模式）
